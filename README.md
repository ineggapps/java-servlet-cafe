## 1. 특징

-   웹 표준 마크업을 지향하고자 HTML5, CSS3를 이용하여 웹 문서를 작성하였습니다.
-   Java 코드에서 자주 사용되는 변수는 인터페이스에 상수로 선언하여 참조하였습니다.
-   Ajax를 이용하여 비동기 통신을 사용하였습니다.
